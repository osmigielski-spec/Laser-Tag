# LASER TAG POZNAŃ - MOODBOARD STYLU GRAFICZNEGO

---

## 1. OBRÓBKA ZDJĘĆ

Zdjęcia powinny oddawać klimat: intensywny, taktyczny, kontrolowany. Ciemne tło, wydobyte akcenty świetlne, bez przesyconej kolorystyki.

### Porównanie stylów:

| ✕ ŹLE | ✓ DOBRZE |
|-------|----------|
| Jasne, przesycone - zbyt wesołe, traci klimat fortu | Ciemne, kontrastowe - głębia, tajemnica, profesjonalizm |
| Niski kontrast - płaskie, nudne, bez charakteru | Wysoki kontrast - głębokie czernie, wyraziste światła |
| Przesycone kolory - zbyt kolorowe, dziecinne | Stonowana paleta + akcent - kontrolowana intensywność |

### Zasady obróbki:

| Parametr | Kierunek |
|----------|----------|
| **Kontrast** | Podkręcony - mocne cienie, wyraziste światło |
| **Saturacja** | Lekko obniżona / stonowana (poza akcentami) |
| **Tło** | Przyciemnione, głębokie |
| **Akcenty świetlne** | Wydobyte (lasery, błyski) - pomarańczowy |
| **Ogólny nastrój** | Intensywny ale kontrolowany, nie chaotyczny |

## 2. ELEMENTY HUD / TAKTYCZNE

Inspiracja grami FPS - celowniki, ramki, wskaźniki. Dodają "gamingowy" charakter bez bycia dziecinnym.

### Elementy do wykorzystania:

| Element | Opis | Zastosowanie |
|---------|------|--------------|
| **Celownik** | Okrąg z krzyżykiem, kropka centralna | Wyróżnienie, fokus na elemencie |
| **Siatka taktyczna** | Linie tworzące grid | Tło, overlay na zdjęciach |
| **Narożniki "bracket"** | Ramki w rogach [ ] | Wyróżnienie zdjęć, sekcji |
| **Wskaźniki / paski** | Poziome paski z wypełnieniem | Statystyki, progress, wyniki |
| **Status "LIVE"** | Kropka + tekst | Dynamika, akcja w czasie rzeczywistym |

### Przykład zastosowania narożników:

```
┌──                              ──┐

        TREŚĆ / ZDJĘCIE

└──                              ──┘
```

## 3. ELEMENTY GRAFICZNE

Geometryczne, ostre kształty. Taktyczna siatka, linie kierunkowe, narożniki "bracket".

### Kształty - TAK:

- **Prostokąty** - ostre rogi, bez zaokrągleń
- **Kształty ścięte** - clip-path z uciętym rogiem (styl militarny)
- **Strzałki / chevron** - kierunkowe, dynamiczne
- **Heksagony** - taktyczne, techniczne

### Kształty - NIE:

- ❌ Zaokrąglone rogi (border-radius)
- ❌ Koła i owale (poza celownikami)
- ❌ Organiczne, "miękkie" formy
- ❌ Dekoracyjne ozdobniki

### Linie kierunkowe:

```
────────────────────────────  (100%)
──────────────────            (70%)
────────────                  (50%)
```

- Gradienty od koloru do przezroczystości
- Wskazują kierunek, budują dynamikę
- Pomarańczowy (#E85D04) jako kolor linii akcentowych

### Zasady użycia koloru akcentowego:

| ✓ TAK | ✕ NIE |
|-------|-------|
| Przyciski CTA | Całe tła sekcji |
| Wyróżnienia w tekście | Duże powierzchnie |
| Linie akcentowe | Każdy element graficzny |
| Ikony aktywne | Nadużywanie "wszędzie" |

**Zasada:** Pomarańczowy tylko na kluczowe elementy - maks 10-15% powierzchni.

## 4. STYL IKON

### Charakterystyka:

- **Liniowe** (outline) lub **płaskie** (solid)
- **Geometryczne**, ostre krawędzie
- **Grubość linii:** 1.5-2px
- **Styl:** Militarny / taktyczny / techniczny

### Przykłady kategorii ikon:

| Kategoria | Styl |
|-----------|------|
| Nawigacja | Proste, geometryczne strzałki |
| Usługi | Ikony tematyczne (tort, ludzie, szkoła) w stylu militarnym |
| Kontakt | Minimalistyczne, liniowe |
| Social media | Standardowe, ale w kolorze marki |

### Kolory ikon:

- **Na ciemnym tle:** Biały (#FFFFFF) lub Pomarańcz (#E85D04)
- **Na jasnym tle:** Granat (#032244) lub Pomarańcz (#E85D04)

## 5. FOTOGRAFIA - WYTYCZNE

### Rodzaje zdjęć:

| Typ | Opis | Obróbka |
|-----|------|---------|
| **Akcja** | Gracze w trakcie gry, ruch, lasery | Wysoki kontrast, wydobyte światła laserów |
| **Sprzęt** | Karabiny, opaski na głowę, wyposażenie | Czyste tło, produktowe, profesjonalne |
| **Fort** | Klimat miejsca, korytarze, teren | Ciemne, atmosferyczne, tajemnicze |
| **Grupy** | Drużyny, uczestnicy przed/po grze | Naturalne, ale z zachowaniem palety |

### Kompozycja:

- **Dynamiczne kadry** - ruch, akcja, ukośne linie
- **Głębia** - wykorzystanie korytarzy fortu, perspektywa
- **Punkt fokusowy** - jeden główny element, reszta w tle
- **Przestrzeń na tekst** - zostawiać miejsce na overlay z hasłami

### Czego unikać w zdjęciach:

- ❌ Prześwietlonych, jasnych zdjęć
- ❌ Chaotycznych kompozycji bez punktu fokusowego
- ❌ "Uśmiechniętych pozówek" (stockowe klimaty)
- ❌ Widocznych elementów niezwiązanych z marką

## 6. CZEGO UNIKAĆ - PODSUMOWANIE

### Elementy ZAKAZANE:

| Kategoria | Unikać |
|-----------|--------|
| **Imprezowe** | Balony, konfetti, serpentyny, "party vibes" |
| **Kolorowe** | Tęcza kolorów, neon, przesycenie, gradient wielokolorowy |
| **Dziecinne** | Gwiazdki, buźki, emotikony, kreskówkowe ilustracje |
| **Zaokrąglone** | Miękkie kształty, duże border-radius, "friendly" design |
| **Stockowe** | Generyczne zdjęcia, uśmiechnięci ludzie przy biurkach |
| **Chaotyczne** | Brak hierarchii, za dużo elementów, bałagan wizualny |

### Dlaczego tego unikamy:

- **Imprezowe/dziecinne** → Nie pasuje do archetypu Władcy, obniża profesjonalizm
- **Kolorowe/neonowe** → Rozmywa paletę marki, traci spójność
- **Zaokrąglone** → Sprzeczne z taktycznym, militarnym charakterem
- **Chaotyczne** → Zaprzecza "pod kontrolą" z esencji marki
